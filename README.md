# Inu FFt 

Example inu app that displays the [`fft`](https://en.wikipedia.org/wiki/Fast_Fourier_transform) of the microphone input.

## how

### install

```
git clone https://github.com/pietgeursen/inu-fft
cd inu-fft
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
